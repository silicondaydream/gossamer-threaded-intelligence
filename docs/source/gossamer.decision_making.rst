gossamer.decision_making package
================================

This package provides distributed decision-making algorithms and voting
mechanisms for multi-agent consensus and preference aggregation.
Agents can leverage these tools to reach collective decisions based on
ranked preferences, approvals, or pairwise comparisons.

Voting methods included:
- Plurality, Borda Count, Approval Voting
- Condorcet winner identification and Schulze (beatpath) method

Use these functions to implement democratic decision processes in swarm
settings or to analyze preference structures under varying protocols.
