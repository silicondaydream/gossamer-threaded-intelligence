gossamer package
===============

The `gossamer` package forms the core of the Threaded Intelligence library.
It aggregates submodules for agent definitions, swarm algorithms,
communication protocols, distributed decision-making, environment perception,
and simulator interfaces.

Use this package as the entry point to access all major Gossamer components.
