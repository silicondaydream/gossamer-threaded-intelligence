gossamer.utils package
======================

Utility functions for computing swarm-level metrics and other helper routines.
These functions help quantify collective behaviors and facilitate analysis of
agent configurations.

Main modules:
- metrics: calculate cohesion, alignment, separation, and other swarm statistics

Extend this package with additional utilities for logging, analysis,
or data processing as needed.
